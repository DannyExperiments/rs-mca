### Domain-uniform random-anchor Bessel theorem

Let (F=\mathbb F_Q), let (D\subset F) consist of (n) distinct points, and let
[
C=RS[F,D,k].
]
Fix (1\le t\le n-k), and put
[
a=k+t,\qquad j=n-a,\qquad N=\binom nj.
]

Let (E\in F[X]) have degree (t) and no zero on (D). Let
[
0\ne B_{\rm num}\in F[X]*{<t},
\qquad
A_E=F[X]/(E),
\qquad
b=[B*{\rm num}]_E\ne0.
]

For (T\in\binom Dj), write (S=D\setminus T). Given (w\in F^D), let (I_S(w)\in F[X]_{<a}) be the polynomial interpolating (w) on (S), and define
[
R_T(w)=\rho_w(T):=[I_S(w)]_E\in A_E.
]
For (z\in F), put
[
\nu_w(z)=
#\left{T\in\binom Dj:\rho_w(T)=zb\right}.
]

Define
[
\lambda=\frac{N}{Q^t},
\qquad
K_r=\binom jr\binom ar,
\qquad
J_t=\sum_{0\le r<t}K_rQ^{-r}.
]

Then:

1. For every (z\in F),
   [
   \mathbb E_w\nu_w(z)=\lambda.
   ]

2. If (T,T'\in\binom Dj) and
   [
   r=|T\setminus T'|=|S\setminus S'|,
   ]
   then
   [
   \operatorname{rank}*F(R_T,R*{T'})
   =t+\min(t,r),
   ]
   and therefore
   [
   \Pr_w!\left[\rho_w(T)=zb,\ \rho_w(T')=zb\right]
   =Q^{-t-\min(t,r)}.
   ]

3. The variance is exactly
   [
   \operatorname{Var}\nu_w(z)
   ==========================

   \lambda
   \sum_{0\le r<t}
   K_r\bigl(Q^{-r}-Q^{-t}\bigr)
   \le \lambda J_t.
   ]

4. Consequently,
   [
   \mathbb E_w\sum_{z\in F}
   \bigl(\nu_w(z)-\lambda\bigr)^2
   \le Q\lambda J_t.
   ]

5. There is a deterministic anchor (w) for which the number (M(w)) of missing slopes satisfies
   [
   M(w)\le \frac{QJ_t}{\lambda}.
   ]
   Moreover, for a uniformly random anchor,
   [
   \Pr_w[\text{some slope is missing}]
   \le\frac{QJ_t}{\lambda}.
   ]

6. The shell factor obeys
   [
   J_t
   \le
   \sum_{r\ge0}\frac{(aj/Q)^r}{(r!)^2}
   \le
   \exp!\left(2\sqrt{\frac{aj}{Q}}\right)
   \le
   \exp!\left(\frac n{\sqrt Q}\right).
   ]

7. The residue datum ((E,B_{\rm num},w)) gives
   [
   \varepsilon_{\rm mca}
   \left(C,1-\frac{k+t}{n}\right)
   \ge
   1-\frac{J_t}{\lambda}.
   ]
   In particular,
   [
   \boxed{\lambda>QJ_t
   \quad\Longrightarrow\quad
   \varepsilon_{\rm mca}
   \left(C,1-\frac{k+t}{n}\right)=1.}
   ]

This is the exact finite theorem. No asymptotic or unproved distributional input is required.

## PROOF_OR_OBSTRUCTION

### 1. Exact pair-rank uses interpolation only

Fix (T,T'), with supports (S=D\setminus T) and (S'=D\setminus T'). Put
[
C_0=S\cap S',
\qquad
r=|S\setminus S'|.
]

For an anchor (w), let
[
P=I_S(w),\qquad P'=I_{S'}(w).
]
Both have degree (<a), and they agree on (C_0). Therefore
[
P-P'=L_{C_0}H
]
for some (H\in F[X]*{<r}), because
[
\deg L*{C_0}=|C_0|=a-r.
]
Thus
[
R_T(w)-R_{T'}(w)
\in [L_{C_0}F[X]_{<r}]_E.
]

Conversely, suppose (u,v\in A_E) and
[
u-v=[L_{C_0}H]*E,\qquad \deg H<r.
]
Choose the degree-(<t) representative (V) of (v), and set
[
P'=V,\qquad P=V+L*{C_0}H.
]
Then (\deg P,\deg P'<a), their residues are (u,v), and they agree on (C_0). They therefore define one word (w) on (S\cup S'), extendable arbitrarily to (D). Hence
[
\operatorname{Im}(R_T,R_{T'})
=============================

\left{(u,v)\in A_E^2:
u-v\in[L_{C_0}F[X]_{<r}]_E\right}.
]

Because (E) has no roots in (D),
[
\gcd(E,L_{C_0})=1,
]
so multiplication by ([L_{C_0}]*E) is invertible. Reduction of (F[X]*{<r}) modulo a degree-(t) polynomial has rank (\min(r,t)). Thus
[
\operatorname{rank}(R_T,R_{T'})
=t+\min(t,r).
]

No use has been made of additive closure, characteristic (p), (D=\mathbb F_p), or any formula involving (X^p-X). This proves the Cycle46 domain-free pair-rank assertion rather than merely assuming it. The prior audit stated precisely this candidate law and identified interpolation as its only mechanism. 【20260619_CYCLE46_TARGETED_WALLBREAKER_AUDIT.md, lines 43–76】

### 2. First and second moments

Each (R_T) is surjective: for any residue (u), take its representative of degree (<t<a) and use its evaluations on (S). Hence (\rho_w(T)) is uniform on the (Q^t)-element space (A_E), giving
[
\mathbb E\nu_w(z)=NQ^{-t}=\lambda.
]

For the pair (T,T'), the target ((zb,zb)) belongs to the pair-map image because its difference is zero. A rank-(d) linear map sends a uniform input to each image point with probability (Q^{-d}), giving the exact joint probability above.

For fixed (T), the number of (T') at exchange distance (r) is
[
K_r=\binom jr\binom ar:
]
remove (r) elements from (T) and insert (r) elements from (D\setminus T). Also
[
\sum_rK_r=N.
]
It follows that
[
\mathbb E\nu_w(z)^2
===================

\lambda\sum_rK_rQ^{-\min(t,r)}.
]
Subtracting
[
\lambda^2
=========

\lambda\sum_rK_rQ^{-t}
]
gives the exact variance formula.

### 3. Bessel shell

For each (r),
[
\binom ar\binom jr
\le
\frac{(aj)^r}{(r!)^2}.
]
Therefore
[
J_t\le\sum_{r\ge0}\frac{(aj/Q)^r}{(r!)^2}.
]
For (y\ge0),
[
\sum_{r\ge0}\frac{y^r}{(r!)^2}
\le
\left(\sum_{r\ge0}\frac{y^{r/2}}{r!}\right)^2
=e^{2\sqrt y}.
]
Finally,
[
2\sqrt{aj}\le a+j=n.
]

This establishes the proposed shell bound exactly.

### 4. Landing in MCA

Suppose (\rho_w(T)=zb), with (S=D\setminus T), and let
[
Q_z=I_S(w).
]
Then
[
Q_z\equiv zB_{\rm num}\pmod E,
]
so
[
P_z=\frac{Q_z-zB_{\rm num}}E
]
is a polynomial of degree (<k). Define the two words
[
f=\frac wE,\qquad
g=-\frac{B_{\rm num}}E
]
pointwise on (D). On (S),
[
f+zg
====

# \frac{w-zB_{\rm num}}E

P_z.
]
Thus (f+zg) agrees with a codeword on the (a=k+t) points of (S).

Noncontainment is automatic from the direction. If (G\in F[X]*{<k}) explained (g) on (S), then
[
EG+B*{\rm num}
]
would vanish on (k+t) points while having degree (<k+t). It would be the zero polynomial, which would imply (E\mid B_{\rm num}). Since

0<\deg E=t,\qquad \deg B_{\rm num}<t,\qquad B_{\rm num}\ne0,

that is impossible.

Therefore every realized slope is support-wise MCA-bad.

## SOURCE_ADMISSIBILITY_CHECK

### 1. Official domain

The packet’s survey restatement defines a smooth domain as a multiplicative coset of a power-of-two-order subgroup. Paper B often uses subgroups, and the packet itself records this subgroup/coset drift. 【proximity_blueprint_v3.tex, lines 100–112】【proximity_blueprint_v3.tex, lines 691–695】

The pair-rank theorem holds for **any set of distinct points**. For the MCA landing, a multiplicative coset lies in (F^\times), so (E=X^t) is nonzero on the domain. Thus the subgroup-to-coset drift causes no mathematical loss.

**Ruling: admissible.**

### 2. (q_{\rm gen},q_{\rm line},q_{\rm chal})

* (q_{\rm gen}): size of the smallest field containing the domain and locator coefficients.
* (q_{\rm line}): size of the field from which (z) is sampled and in which the line experiment is analyzed.
* (q_{\rm chal}): not defined in the packet as a mathematical quantity. The packet only warns that it need not equal (q_{\rm line}).

For the scalar MCA theorem,
[
Q=q_{\rm line}=|F|.
]
In the same-field specialization,
[
q_{\rm gen}=q_{\rm line}=|F|.
]
No statement involving (q_{\rm chal}) follows without a protocol reduction.

**Ruling: theorem admissible; any protocol-level identification with (q_{\rm chal}) requires an extra hypothesis.**

### 3. Arbitrary anchors

The residue-line definition explicitly says
[
w:D\to F
]
is an arbitrary anchor word. It does not require (w) to be polynomial, base-valued, monic, or canonical. 【slackMCA_v3.tex, lines 1187–1195】

The origin audit likewise says arbitrary anchors are part of the definition and that restricting to monic anchors does not cover the full problem. 【paper_b_counterexample_comparison.md, lines 149–194】

**Ruling: random-anchor selection is literally admissible.**

### 4. Quadratic extension alphabet

The proof is over one arbitrary finite field (F=\mathbb F_Q). It does not require
[
\mathbb F_q\subset\mathbb F_{q^2}.
]
The Cycle45 half-entropy threshold arose from its particular ledger
[
n=p,\qquad q_{\rm line}=p^2,
]
not from the pair-rank lemma.

For an official code (RS[F,L,k]), simply run the theorem over that same (F). In particular, setting (\mathbb B=F) removes every extension issue.

**Ruling: no field mismatch.**

### 5. Extension-coordinate transfer

When (L\subseteq\mathbb B\subsetneq F), coordinate expansion gives
[
RS[F,L,k]\cong RS[\mathbb B,L,k]^e,
]
and an (F)-line becomes the structured multiplication slice
[
\Phi(f)+M_z\Phi(g).
]
The source proves this equivalence exactly. 【f1_extension_coordinate_transfer.md, lines 5–63】

This does **not** make the result non-scalar in the official alphabet-(F) formulation. It means the same scalar (F)-construction has an equivalent interleaved-(\mathbb B) representation. What would be invalid is to call it ordinary scalar MCA for (RS[\mathbb B,L,k]), or to infer it from a base-field scalar numerator.

**Ruling: scalar for the official (F)-code; multiplication-slice interleaved when rewritten over (\mathbb B).**

### 6. Minimal denominator and shared factors

The Frobenius-compressed construction has
[
E=H(X)(X-\theta),\qquad B_{\rm num}=H(X),
]
so
[
\gcd(E,B_{\rm num})=H.
]
Its rational direction reduces to denominator degree one, which is why the packet calls it a possible packaging artifact. 【20260619_CYCLE46_INSTANCE3_FROBENIUS_COMPRESSED_AUDIT.md, lines 137–169】

For the Bessel route, choose
[
E=X^t,\qquad B_{\rm num}=1.
]
Then
[
\gcd(E,B_{\rm num})=1.
]
Thus even a strengthened reduced-denominator convention would not remove the construction.

**Ruling: nonminimality matters only to the Frobenius-compressed route, not to Bessel.**

### 7. Remaining literal-official source obstruction

The packet explicitly lists as unfinished the alignment between the paper’s support-wise MCA definition and the survey’s formal Definition 4.3, including quantifiers and finite-length rounding. The formal survey definition is absent from the packet. 【proximity_blueprint_v3.tex, lines 631–640】
