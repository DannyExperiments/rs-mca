# Experimental Data

This directory contains compact reproducibility artifacts:

- `certificates/`: JSON certificates and certificate-specific READMEs.
- `schemas/`: JSON schemas used by certificate emitters.
- `witnesses/`: explicit witness fixtures.

Keep generated bytecode, raw model transcripts, temporary run logs, and bulky
archives out of this tree. If a large artifact is needed, add a compact note in
`notes/` explaining how to regenerate it.
